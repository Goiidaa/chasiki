## Installation

> [как это скачать]

1.копируйте рипозиторий
 ```
 git clone https://github.com/dantas-arthur/ClockTemp.git
 ```

2. откройте
  ```
  cd ваша/диерктория/ClockTemp/script
  ```
3. запустите этот код для устоновки
 ```
 sudo ./install.sh
 ```
4.  запустите програму с помощью команды  clocktemp
```
 clocktemp
 ```
